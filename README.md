# Projectes de DAW, orientats a **Experiències interactives**
Recull dels projectes transversals i finals de DAW de l'Institut Pedralbes

## 2025 - 2026
### TR1 - Type Racer Royale 

Realitzat durant ´tres setmanes entre octubre i novembre, en grups d'uns 4 alumnes. 

**Objectiu:** Desenvolupar una aplicació web completa d'un joc de mecanografia multijugador en temps real. L'objectiu és aplicar els coneixements de Vue.js per al frontend i Node.js amb Socket.IO per al backend, creant una experiència interactiva i dinàmica des de zero fins al seu desplegament final amb Docker.

#### Tecnologies clau

- **Frontend:** Vue.js 3 (Composition API amb `<script setup>`)
- **Backend:** Node.js amb la llibreria `socket.io`
- **Desplegament:** Docker i Docker Compose

---

#### Concepte del joc

"Type Racer Royale" és una batalla de mecanografia. Diversos jugadors s'uneixen a una sala. Quan comença la partida, a tots els apareix una pila de paraules. A mesura que completen paraules correctament, envien "paraules de penalització" als seus oponents, fent que les seves piles creixin. Si la pila de paraules d'un jugador arriba a un límit màxim, queda eliminat. L'últim jugador que quedi en peu guanya la partida.
